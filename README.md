# 🚀 Produtividade App

Um aplicativo de alta performance para gestão de hábitos e tarefas diárias, desenvolvido com **React** e **Vite**. O foco é simplicidade, design moderno (Glassmorphism) e funcionamento offline (PWA).

## ✨ Funcionalidades

-   **✅ Gestão de Tarefas:** Adicione, conclua e exclua tarefas com níveis de prioridade (Alta, Média, Baixa).
-   **🔄 Monitoramento de Hábitos:** Acompanhe seus hábitos diários com indicadores visuais e cores personalizadas.
-   **📊 Relatórios Inteligentes:**
    -   **Círculos de Progresso:** Visualização rápida da performance do dia, semana e mês.
    -   **Mapa de Calor (Heatmap):** Acompanhamento mensal inspirado no gráfico de contribuições do GitHub.
    -   **Sequência (Streaks):** Contador de dias consecutivos mantendo o foco.
-   **📱 PWA (Progressive Web App):** Instalável no celular e funciona offline via Service Workers.
-   **🌙 Interface Dark Mode:** Design otimizado para reduzir o cansaço visual, com efeitos de blur e gradientes.

## 🛠️ Tecnologias Utilizadas

-   **React 18** (Hooks, LocalStorage)
-   **Vite** (Build tool ultra-rápida)
-   **CSS-in-JS** (Estilização dinâmica e responsiva)
-   **Service Workers & Web Manifest** (Funcionalidades PWA)

## 🚀 Como Rodar o Projeto

1.  **Clone o repositório:**
    ```bash
    git clone [https://github.com/alissoncardoso-afk/Produtividade-App.git](https://github.com/alissoncardoso-afk/Produtividade-App.git)
    ```
2.  **Instale as dependências:**
    ```bash
    npm install
    ```
3.  **Inicie o servidor de desenvolvimento:**
    ```bash
    npm run dev
    ```
4.  **Para gerar a versão de produção:**
    ```bash
    npm run build
    ```

## 📦 Deploy

Este projeto está configurado para deploy automático na **Vercel**. Cada push para a branch `main` gera uma nova versão estável.
